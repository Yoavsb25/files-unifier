## Overview

PDF Batch Merger is a Python desktop application designed for high-volume document assembly. It lets you select a folder of PDF files and merge them into a single output document, maintaining order and handling edge cases like corrupt or empty files.

## Features

- Batch merge an entire folder of PDFs into one file
- Configurable output file name and destination
- Handles large volumes efficiently
- Simple desktop interface — no browser required

## Getting Started

**Prerequisites:**
- Python 3.x
- pip

**Installation:**
```bash
git clone https://github.com/Yoavsb25/files_unifeder.git
cd files_unifeder
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```

Select your source folder and output path in the desktop UI, then click Merge.
